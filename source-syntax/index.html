 (require syntax/source-syntax) package： source-syntax

函数

(recover-source-syntax orig 
  expanded 
  [#:traverse-now? now?]) 
 → (-> syntax? (or/c syntax? #f))
  orig : syntax?
  expanded : syntax?
  now? : boolean? = #f
Return a procedure that accepts a syntax object from expanded and returns the outermost syntax object in orig that has the same location as the given syntax object. If no syntax object in orig has the same location as the given syntax object, the procedure repeats with the parent of the given syntax object.

Example:
> (let* ([orig #'(λ (x [y 0]) (+ x y))]
         [expanded (expand orig)]
         [recovered ((recover-source-syntax orig expanded) expanded)])
    (syntax? recovered))

#t
